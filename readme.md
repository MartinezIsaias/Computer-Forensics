This is a project for Computer Forensics class at the University of Louisville. The provided tsv file was mounted using FTK Imager to conduct a mock forensis analysis of the files that existed on a computer to acquire details to solve a crime. I used software such as: JLECMD, LECmd, PECmd, EVTXEcmd, $I_Parse, RegistryExplorer, PrefetchParse, TimelineExplorer, EventLogParse, and forensic analysis skills to gather as much evidence as possible. The .docx file's contents is a summarization of my findings, evidence to support claims, and methods of analysis.
